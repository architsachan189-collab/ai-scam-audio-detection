\# AI-Based Scam Detection using Audio Features



\## Overview

This project explores how audio signals can be used to identify potential scam calls. It focuses on building a simple machine learning pipeline that classifies speech as “scam” or “not scam” based on voice characteristics.



\## Dataset

The model is trained using the RAVDESS Emotional Speech Dataset, which contains labeled speech recordings with different emotional expressions.



\## Approach

\- Processed raw audio files and extracted MFCC (Mel-Frequency Cepstral Coefficients) features

\- Built a classification model using Logistic Regression

\- Achieved approximately 72% accuracy on test data



\## Pipeline

Audio Input → Feature Extraction → Model Training → Prediction



\## Current Status

This is an ongoing project. The current version focuses on building a working pipeline, and further improvements are being made to use more realistic datasets and enhance model performance.
